# İş İlanı Karar Destek Sistemi

CV'leri iş ilanlarıyla anlamsal olarak eşleştirip skorlayan bir Django uygulaması.

## Ne İşe Yarıyor

- İşverenler ilan girebiliyor, adaylar CV/başvuru yükleyebiliyor
- Sistem CV içeriğini anlamsal olarak analiz edip ilanla ne kadar uyumlu olduğunu ölçüyor
- Eşleşmelere bir skor veriyor, böylece en uygun adaylar öne çıkıyor

## Kullanılan Teknolojiler

- Python / Django
- HTML / CSS

## Kurulum

```
git clone https://github.com/nazyi/Is_Ilan_KararDestekSistemi.git
cd Is_Ilan_KararDestekSistemi
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```
